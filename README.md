## 它到底是怎么运作的？

流程非常简单，三步搞定：

1. 注册账号，充值余额（最低 $1 起充）
2. 选择目标国家，选择你要接收验证码的网站或 App
3. 系统分配一个临时号码给你，把号码填到注册页面，验证码会直接显示在你的账户里

号码租赁时长是 20 分钟，期间可以无限接收来自同一网站的短信（重复短信享 9 折优惠）。如果号码没收到短信，可以直接取消，余额不扣。

整个过程不需要 SIM 卡，不需要实体手机，账号里操作就行。
